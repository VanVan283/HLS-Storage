Custom JW variable guide
Custom JW Full uses runtime variables window.VV_*, customer writes JS to handle player/sub/multi.
Biến runtime có sẵn
• window.VV_SRC: link video hiện tại
• window.VV_SUB_URL: link subtitle hiện tại
• window.VV_TITLE: tiêu đề video
• window.VV_VAST_URL: link VAST hiện tại
• window.VV_VMAP_URL: link VMAP hiện tại
• window.VV_MULTI_SOURCES: mảng nhiều nguồn/server
• window.VV_CURRENT_INDEX: index nguồn hiện tại
• window.VV_CURRENT_LABEL: label nguồn hiện tại
Ví dụ tối thiểu
var p = jwplayer("jw-player");
p.setup({
  file: window.VV_SRC,
  tracks: window.VV_SUB_URL ? [
    { file: window.VV_SUB_URL, kind: "captions", label: "Sub", "default": true }
  ] : [],
  controls: true,
  stretching: "uniform"
});
Ví dụ multi source
var list = Array.isArray(window.VV_MULTI_SOURCES) ? window.VV_MULTI_SOURCES : [];
var current = list[window.VV_CURRENT_INDEX] || list[0] || null;

jwplayer("jw-player").setup({
  file: current && current.url ? current.url : window.VV_SRC,
  controls: true
});
Hướng dẫn biến Custom Player
Cách dùng SRC / ads / watermark trong template custom
1) Biến cơ bản
• {{SRC}}: link video / HLS để player custom phát video
• {{SUB_URL}}: link subtitle tương ứng của video hiện tại (nếu có)
• {{TITLE}}: tiêu đề video hiện tại
• {{VAST_URL}}: link VAST hiện đang cấu hình
• {{VMAP_URL}}: link VMAP hiện đang cấu hình
• {{MULTI_SOURCES_JSON}}: danh sách source server cho multi embed (JSON)
• {{MULTI_CURRENT_INDEX}}: index server hiện tại (multi embed)
• {{MULTI_CURRENT_LABEL}}: nhãn server hiện tại (multi embed)
2) Biến watermark theo domain
• {{WATERMARK_ENABLED}}: 1 nếu domain hiện tại có watermark, ngược lại là 0
• {{WATERMARK_LOGO}}: URL logo watermark
• {{WATERMARK_POSITION}}: vị trí watermark, ví dụ top-right, top-left, center
• {{WATERMARK_MODE}}: chế độ hiển thị, ví dụ always hoặc first15
• {{WATERMARK_SECONDS}}: số giây watermark hiện nếu dùng mode theo thời lượng
3) Ví dụ nhanh
<div id="wrap" style="position:relative;width:100%;height:100vh;background:#000">
  <video id="v" src="{{SRC}}" controls playsinline style="width:100%;height:100%">
    <track kind="subtitles" srclang="vi" label="Sub" src="{{SUB_URL}}" default>
  </video>
  <img id="wm" src="{{WATERMARK_LOGO}}" style="position:absolute;max-width:120px;display:none" />
</div>
<script>
  const wm = document.getElementById('wm');
  const pos = '{{WATERMARK_POSITION}}';
  const mode = '{{WATERMARK_MODE}}';
  const secs = Number('{{WATERMARK_SECONDS}}' || 15);
  const enabled = '{{WATERMARK_ENABLED}}' === '1';

  if (enabled && wm) {
    if (pos === 'top-left') { wm.style.top='12px'; wm.style.left='12px'; }
    else if (pos === 'center') { wm.style.top='50%'; wm.style.left='50%'; wm.style.transform='translate(-50%,-50%)'; }
    else { wm.style.top='12px'; wm.style.right='12px'; }

    if (mode === 'always') {
      wm.style.display = 'block';
    } else {
      wm.style.display = 'block';
      setTimeout(() => wm.style.display = 'none', secs * 1000);
    }
  }
</script>
4) Ví dụ ads VAST / VMAP
<script>
  const vast = '{{VAST_URL}}';
  const vmap = '{{VMAP_URL}}';

  if (vmap) {
    console.log('Dùng VMAP:', vmap);
    // truyền vmap vào ad SDK / player custom của anh
  } else if (vast) {
    console.log('Dùng VAST:', vast);
    // truyền vast vào ad SDK / player custom của anh
  }
</script>
Lưu ý: template custom chỉ nhận biến và tự xử lý hiển thị. Nếu muốn hiện subtitle thì chèn thẻ <track src="{{SUB_URL}}" ...> vào player custom. Nếu muốn logo/ads hiện đẹp thì phần HTML/CSS/JS của khách phải dùng các biến này.
